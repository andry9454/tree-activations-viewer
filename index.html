Syntactic Tree Viewer with Activations!

This is a d3.js-based tree viewer that allows you to expand and collapse tree nodes and pan/zoom to different areas of the tree.
In allows also to see activations levels per subtrees. Input the bracketed tree that you want to view in the box above and press "View Tree". The viewer has been designed to work with classical LISP style trees written like TAG_NAME:ACTIVATION. Example: (S:0.5 (NP:0.6 this pippo) (VP:0.1 (V:0.2 is) (AdjP:0.3 pretty)))
